## Objectifs

Pédagogiquement, ce TP a pour but de :

- vous faire découvrir l'architecture frontend en utilisant des **composants** ;
- vous faire découvrir des notions de **Javascript ES6** et **TypeScript** ;
- vous faire découvrir les bases de la librairie **React.js**.

Dans ce TP, vous allez commencer à implémenter l'interface de votre jeu de touché-coulé. Notamment les parties suivantes :

- implémentation de la vue d'ajout des bateaux ;
- implémentation de la page d'authentification ;
- début de l'implémentation du composant **Carte**.
